模块化（避免污染全局变量，团队协作不会产生冲突）

  • 单例模式 var obj={} 名字无法避免冲突
  • 闭包 seajs CMD 就近依赖 requirejs AMD 依赖前置
  • common js规范 定义了如何使用模块 如何定义模块 如何导出模块
    • 如何定义模块 在node文件中每个js就是一个模块，为了每个模块之间没有关系 所以在代码执行时外层套了一个闭包
    • 形参 exports require module –filename __dirname
    • 在闭包中 将this指向改变为exports，为空对象

    安装

  • 全局安装 -g （只能在命令行下使用）

    • nrm帮助我们切换安装的源头

    • 全局安装不会配置到环境变量中，由于npm已经在环境变量中了，相当于在npm下有一个快捷方式，链接真实的文件
